## 二、API 调用

### 2.1 获取插件列表
```python
# 社区 API 基础URL
BASE_URL = "https://community.nekro.ai"

# 获取插件列表
GET /api/plugin?page=1&pageSize=100

# 获取插件详情
GET /api/plugin/{module_name}

# 获取仓库信息
GET /api/plugin/{module_name}/repo
```

### 2.2 调用示例
```python
import httpx

async def get_all_plugins():
    """获取所有插件"""
    async with httpx.AsyncClient(base_url="https://community.nekro.ai") as client:
        # 第一页，获取100个
        response = await client.get("/api/plugin?page=1&pageSize=100")
        return response.json()
```
